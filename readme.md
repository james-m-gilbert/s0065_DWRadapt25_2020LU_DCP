
## s0065_DWRadapt25_2020LU_DCP

Version of DWR's 2025 climate adaptation scenario (CCA6_v3.8.0) with the Delta Conveyance Project (DCP), adapted for use in the library of COEQWAL scenarios.
The basis for this scenario is the DWR version CCA6_v3.8.0_A_2043_50CC_LU100_SLR15_20250515 as made available via the Downloads tab at https://cap-recon.azurewebsites.net/ (direct link: https://cadwr.box.com/s/7832s6jtrky11mq7wrss7hazltxgmmv2?download=1). 
The SV file was modified to include demands based on 2020 land use, with modifications for the healthy rivers and landscapes (formerly Voluntary Agreements) rice acreage fallowing in the Sacramento Valley and to use adjusted historic hydrology (rather than the CC50 in the original DWR scenario).
